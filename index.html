Projects

Color Game
Web Game
HTML5, CSS3, JavaScript

This project is quite simple and fun. You are given a random RGB value. That RGB value is then passed onto 1 of 6 circles (3 if on easy). You must choose which RGB value you think is the correct one. Once you choose the correct one, all other cirlces and the header turn into that same color. You can play the game again by pressing "Play Again?". This project was done as a follow along to Steele Colt's course on JavaScript developing which can be found at Udemy.
